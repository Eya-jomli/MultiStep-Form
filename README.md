# MultiStep Registration Form

This project is a **multi-step registration form** built with Laravel.  
It contains **7 steps** for user registration, validating inputs at each step.

## Demo
[Live Demo Link]([your-demo-link-here](https://youtu.be/3vLqIsPf324))

## Features
- Multi-step form (7 steps)
- Input validation
- Database integration (MySQL)
- Responsive design
